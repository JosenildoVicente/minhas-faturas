## Descrição do Projeto
Um projeto em Python de gerenciamento de faturas e contas

### Futuras features

 - Cadastro de usuários
    - Saldo atual
    - Receitas total
    - Despesas total
    - Receitas do mês
    - Despesas do mês
    - Contas
        - Separação de tipos de contas
            - Corrente
            - Poupança
            - Pagamentos
    - Faturas de cartões de crédito
        - Detalhes
        - Opção de colocar nomes nos itens da fatura
